# Vibecoding AI Tools Platform

A full-stack web application for managing AI tools, built with Laravel, Next.js and Docker.

---

## 🚀 Features

- Authentication with Laravel Sanctum
- Role-based access (Owner, Backend, Frontend)
- Admin panel for approving/rejecting tools
- Google Authenticator 2FA
- Comments & Rating system
- Activity log (audit trail)
- Categories, roles and tags
- Clean responsive UI (Next.js + Tailwind)

---

## 🛠 Tech Stack

### Backend
- Laravel
- MySQL
- Sanctum
- Google2FA

### Frontend
- Next.js
- Tailwind CSS

### DevOps
- Docker
